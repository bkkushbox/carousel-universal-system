# Test Scenarios

## 1. Content-only
`Нужно создать карусель на тему как перестать откладывать задачи, без моих фото, нужен только текст.`

Ожидание:
- система не просит data verification;
- задаёт уточняющие вопросы по аудитории, цели и тону.

## 2. Data-first
`Нужно создать карусель на тему AI-агентов, показать реальные сравнения по стоимости и возможностям, нужен полный пакет.`

Ожидание:
- система включает research + verification;
- не выдумывает цифры;
- помечает, что перепроверить перед публикацией.

## 3. Photo-based
`Нужно создать карусель на тему личного бренда, использовать мои загруженные фотографии, сделать визуально сильную обложку и тексты по слайдам.`

Ожидание:
- система включает photo-based mode;
- спрашивает про роль фото;
- определяет safe zones.

## 4. Mixed mode
`Нужно создать карусель на тему DeFi, показать реальные сравнения прибыли, использовать мои загруженные фотографии, сначала задай уточняющие вопросы.`

Ожидание:
- система понимает, что нужны и verification, и photo mode;
- задаёт предметные вопросы;
- не генерирует карусель сразу.

## 5. System mode
`Помоги собрать универсальный project для каруселей, который можно загрузить в ChatGPT Project.`

Ожидание:
- система переключается в system mode;
- предлагает instructions, templates, start prompt и checklist.
