# Carousel Universal System

Универсальный репозиторий для ChatGPT Project, который помогает собирать Instagram-карусели почти на любую тему.

Это не нишевый prompt под одну тему и не клон чужого репозитория. Это свой продукт с упором на:
- intake перед генерацией;
- умную маршрутизацию по типу задачи;
- photo-based режим;
- verification layer для данных;
- anti-slop редактуру;
- project-ready упаковку.

## Для чего это

Система нужна, когда пользователь хочет:
- сделать карусель на любую тему;
- использовать реальные данные и сравнения;
- использовать загруженные фото;
- получить не просто текст, а структурный production package.

## Как использовать

### Вариант 1. Через ZIP из GitHub
1. Скачать репозиторий как ZIP.
2. Открыть ChatGPT Project.
3. Загрузить ZIP или файлы из него в Project.
4. Использовать файлы из папки `project/` как основу для Instructions и первого запуска.

### Вариант 2. Через файлы
Загрузить в Project:
- `SKILL.md`
- `project/CHATGPT_PROJECT_INSTRUCTIONS.txt`
- `project/START_HERE.txt`
- `references/OUTPUT_TEMPLATE.md`
- `references/REFERENCE_CHECKLIST.md`

## Что лежит внутри

- `SKILL.md` – ядро поведения системы
- `START_HERE.md` – с чего начать человеку
- `references/` – шаблоны и чек-листы
- `project/` – готовые файлы для ChatGPT Project
- `examples/` – примеры пользовательских запросов

## Как должен выглядеть первый запрос

Пример:

`Нужно создать карусель на тему DeFi, вывести реальные сравнения прибыли, использовать мои загруженные фотографии, сначала задай уточняющие вопросы.`

После этого система должна:
1. определить тип задачи;
2. увидеть, что нужны и data verification, и photo mode;
3. задать уточняющие вопросы;
4. собрать verified result;
5. выдать структурный пакет.

## Чем этот репозиторий отличается от photo-only подходов

Этот репозиторий не ограничен cover photo и не завязан на одну нишу.

Он рассчитан на:
- data-heavy темы;
- photo-heavy темы;
- mixed mode сценарии;
- универсальные темы, а не только финансы или личные фото;
- project-system use case, а не только отдельную генерацию HTML.
