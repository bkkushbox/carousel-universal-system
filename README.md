# Carousel Universal System

Универсальный навык для Instagram-каруселей почти на любую тему.

Подходит для задач, где нужно:
- сначала задать уточняющие вопросы;
- при необходимости проверить факты и цифры;
- при необходимости использовать загруженные фото;
- собрать не просто текст, а аккуратный рабочий пакет.

## Как использовать

### 1. Добавьте навык по ссылке на GitHub

Если в вашем интерфейсе Codex есть кнопка **«Добавить навык»**, используйте ссылку:

`https://github.com/bkkushbox/carousel-universal-system`

### 2. Дайте первый тестовый запрос

Пример:

`Нужно создать карусель на тему DeFi, показать реальные сравнения прибыли, использовать мои загруженные фотографии, сначала задай уточняющие вопросы.`

Правильное поведение:
- навык не генерирует карусель сразу;
- определяет тип задачи;
- понимает, что нужны и фото, и проверка данных;
- задаёт уточняющие вопросы;
- только потом строит результат.

### 3. Если хотите работать в отдельном Project

После установки навыка можно создать отдельный Project для каруселей и хранить там:
- фото;
- референсы;
- файлы по теме;
- историю одной контентной системы.

## Что внутри

- `SKILL.md` – поведенческое ядро навыка
- `skill.json` – metadata
- `agents/openai.yaml` – конфигурация загрузки
- `references/OUTPUT_TEMPLATE.md` – шаблон финального вывода
- `references/REFERENCE_CHECKLIST.md` – чек-лист по фото, данным и референсам
- `examples/TEST_SCENARIOS.md` – тестовые сценарии
- `project/` – вспомогательные файлы для использования внутри Project

## Какие сценарии навык покрывает

- обычная контентная карусель;
- карусель с цифрами и сравнениями;
- карусель с пользовательскими фото;
- mixed mode: реальные данные + фото;
- story-карусель;
- sales-карусель;
- case-study;
- сборка шаблона или системы вместо готовой карусели.

## Короткие примеры запросов

**Data-first**

`Нужно создать карусель на тему AI-агентов, показать реальные сравнения по стоимости и возможностям, без моих фото, нужен полный пакет.`

**Photo-based**

`Нужно создать карусель на тему личного бренда, использовать мои загруженные фотографии, сделать сильную обложку и тексты по слайдам.`

**Mixed mode**

`Нужно создать карусель на тему DeFi, показать реальные сравнения прибыли, использовать мои загруженные фотографии, сначала задай уточняющие вопросы.`

## Если что-то пошло не так

Если навык вместо работы начинает обсуждать свою структуру или уходить в мета-режим, запустите новый чистый чат и дайте прямой пользовательский запрос без обсуждения репозитория.
