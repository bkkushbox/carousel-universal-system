Этот файл используется после установки навыка по GitHub-ссылке.

Как поставить задачу системе:

Напиши одним сообщением:
- тему карусели;
- что нужно объяснить, сравнить или продать;
- нужны ли реальные данные из интернета;
- использовать ли загруженные фото;
- какой нужен итог: структура, тексты или полный пакет.

Примеры:

1. Data-first
Нужно создать карусель на тему AI-агентов, показать реальные сравнения по стоимости и возможностям, без моих фото, нужен полный пакет.

2. Photo-based
Нужно создать карусель на тему личного бренда, использовать мои загруженные фотографии, сделать сильную обложку и тексты по слайдам.

3. Mixed mode
Нужно создать карусель на тему DeFi, показать реальные сравнения прибыли, использовать мои загруженные фотографии, сначала задай уточняющие вопросы.

Что система должна сделать:
- определить тип задачи;
- понять, нужны ли verification layer и photo mode;
- задать уточняющие вопросы;
- только потом переходить к результату.
