# Start Here

Это skill-first репозиторий.

## Правильный порядок

1. Установить навык по GitHub-ссылке
2. После установки создать отдельный Project
3. Уже внутри Project давать пользовательские задачи

## Что написать для установки

```text
Установи этот навык: https://github.com/bkkushbox/carousel-universal-system
```

## Что делать после установки

Создай отдельный Project для каруселей и используй там этот навык.

## Как ставить задачу

Напиши одним сообщением:
- тему карусели;
- что нужно объяснить, сравнить или продать;
- нужны ли реальные данные из интернета;
- использовать ли загруженные фото;
- какой нужен итог: структура, тексты или полный пакет.

## Примеры

### Data-first
`Нужно создать карусель на тему AI-агентов, показать реальные сравнения по стоимости и возможностям, без моих фото, нужен полный пакет.`

### Photo-based
`Нужно создать карусель на тему личного бренда, использовать мои загруженные фотографии, сделать сильную обложку и тексты по слайдам.`

### Mixed mode
`Нужно создать карусель на тему DeFi, показать реальные сравнения прибыли, использовать мои загруженные фотографии, сначала задай уточняющие вопросы.`

## Что система должна сделать

Она не должна сразу писать карусель.

Она должна:
1. определить тип задачи;
2. понять, нужны ли verification layer и photo mode;
3. задать уточняющие вопросы;
4. только потом переходить к результату.
